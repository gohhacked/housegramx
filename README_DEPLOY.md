# HouseGramX

Custom Telegram Web Client

## Setup

1. Install dependencies: `npm install`
2. Copy `.env.example` to `.env`
3. Add your Telegram API credentials from https://my.telegram.org
4. Run: `npm run dev`

## Deploy

### Vercel
1. Import repository
2. Add environment variables
3. Deploy
