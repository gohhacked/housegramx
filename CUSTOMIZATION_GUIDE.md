# 🎨 Руководство по кастомизации HouseGramX

## Как открыть настройки

1. Открой меню (три точки слева вверху)
2. Нажми "🎨 Кастомизация"
3. Откроется окно с 4 вкладками

## Вкладка "Темы"

### Готовые темы:
- **Океан** 🌊 - синие тона с волнами
- **Закат** 🌅 - оранжево-розовый градиент
- **Лес** 🌲 - зелёные тона с частицами
- **Полночь** 🌙 - фиолетовые тона со звёздами
- **Киберпанк** 🤖 - неоновые цвета с эффектом матрицы

Просто нажми на тему чтобы применить!

## Вкладка "Фон"

### Загрузка своего фона:
1. Нажми "📁 Выбрать файл"
2. Выбери картинку или GIF
3. Фон автоматически применится

### Живые обои:
- **Волны** - плавные движущиеся волны
- **Частицы** - летающие частицы
- **Звёзды** - мерцающие звёзды
- **Матрица** - падающие символы как в фильме

## Вкладка "Шрифт"

Доступные шрифты:
- По умолчанию (Roboto)
- Inter - современный геометрический
- Poppins - округлый и дружелюбный
- Montserrat - элегантный
- Open Sans - универсальный
- Lato - лёгкий и читаемый
- Raleway - утончённый
- Nunito - мягкий и приятный

## Вкладка "Режимы"

### 📐 Компактный режим
- Уменьшенные отступы
- Больше контента на экране
- Меньше высота сообщений
- Идеально для маленьких экранов

### 🎯 Режим фокуса
- Скрывает кнопки меню
- Убирает правую колонку
- Скрывает папки чатов
- Минимум отвлекающих элементов
- Только чаты и сообщения

## Технические детали

### Где хранятся настройки?
- В localStorage браузера
- Ключ: `housegramx_customization`
- Сохраняются автоматически

### Формат данных:
```json
{
  "activeTheme": "ocean",
  "customThemes": [],
  "font": "default",
  "compactMode": false,
  "focusMode": false
}
```

### Создание своей темы:
```javascript
{
  "name": "Моя тема",
  "colors": {
    "primary": "#0088cc",
    "secondary": "#00a8e8",
    "background": "#0a1929",
    "surface": "#1a2332",
    "text": "#ffffff",
    "textSecondary": "#b0bec5",
    "accent": "#00d4ff",
    "error": "#ff5252",
    "success": "#4caf50",
    "warning": "#ff9800"
  },
  "chatBackground": {
    "type": "gradient",
    "value": ["#0088cc", "#00a8e8"]
  }
}
```

## Советы

1. **Для работы** - используй компактный режим
2. **Для чтения** - включи режим фокуса
3. **Для красоты** - попробуй живые обои
4. **Для глаз** - выбери тёмную тему
5. **Для скорости** - отключи анимации

## Горячие клавиши

- `Ctrl + ,` - открыть настройки (планируется)
- `Ctrl + Shift + K` - компактный режим (планируется)
- `Ctrl + Shift + F` - режим фокуса (планируется)

## Проблемы?

Если что-то не работает:
1. Очисти кэш браузера (Ctrl+Shift+Delete)
2. Перезагрузи страницу (Ctrl+F5)
3. Проверь консоль (F12)
4. Сбрось настройки (удали ключ из localStorage)
